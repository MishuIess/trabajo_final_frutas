## Trabajo Final - Analisis de datos de frutas preferidas o favoritas

El presente trabajo final tiene como finalidad crear y analizar los datos del archivo . csv y .sqlite.

Las herramientas utilizadas para el trabajo final de frutas es pandas, SQLite, Matplotlib y Google Colab.

Descripción de los pasos realizados en el proyecto: 

1.- Importación de librerías necesarias como Pandas, Matplotlib y SQLite.

2.- Crea el archivo .csv y se crea también el archivo .sqlite

3.- Carga del dataset CSV utilizando la función read_csv().

4.- Conexión a la base de datos SQLite mediante sqlite3.connect().

5.- Lectura de información desde la tabla Answer utilizando consultas SQL.

6.- Exploración de los datos mediante funciones como head(), info() y describe().

7.- Limpieza de datos eliminando registros nulos.

8.- Integración de ambos datasets utilizando la función merge() de Pandas.

9.- Generación de visualizaciones estadísticas con Matplotlib.

10.- Exportación del dataset final procesado.

11.- Organización y subida de archivos al repositorio GitHub.
